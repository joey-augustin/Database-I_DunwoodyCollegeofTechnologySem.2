# Final - Sushi Bar

## Overview
This project involved designing and implementing a relational database for Crescent Sushi Bar, a fictional restaurant offering online ordering and dinner reservations. The database was built to support core business operations including customer management, menu browsing, payment processing, order tracking, and reservation handling. Many table rows (menu items, prices) were established by the professor who designed the HiFi website.

## What I Learned
- Designing normalized relational schemas (3NF)
- Enforcing referential integrity with primary and foreign keys
- Writing DDL, DML, and DQL in T-SQL
- Using views, indexes, and roles to organize and secure a database

## Technologies
- SQL Server (SSMS)
- T-SQL
